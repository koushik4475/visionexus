3. Screenshot Capture Guidelines
Optimal Settings:

Resolution: 1920x1080 or higher
Format: PNG for UI (lossless), JPG for photos
Quality: High DPI for sharp text
Browser: Use for web components at 100% zoom

Capture Checklist:

 Ensure all UI elements are visible
 Include sample data in forms/tables
 Show active states (detection boxes, alerts)
 Capture during peak visual activity
 Include mouse cursor for interactive elements
